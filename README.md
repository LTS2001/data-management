# AI Labs 项目

基于 `@umijs/max` 构建的实验性对话系统。

## 项目简介

AI Labs 是一个用于管理和评估 AI 对话的平台，支持创建对话、发送消息、查看历史记录以及评估对话质量。

## 主要功能

### 实验管理

- 实验列表查看
- 创建新实验
- 查询现有实验

### 对话系统

- 创建新对话
- 查询现有对话
- 发送消息
- 查看对话历史
- 消息编辑功能

### 评估系统

- 对消息进行评分
- 查看评估结果
- 分析对话质量

## 技术栈

- **UI 组件**：Ant Design、Ant Design X
- **状态管理**：React Hooks
- **国际化**：i18next
- **构建工具**：UmiJS Max
- **类型检查**：TypeScript

## 实验功能使用说明

### 创建新对话

1. 进入实验页面
2. 点击"创建新对话"按钮
3. 填写相关信息（联系人 ID、Union ID、评分等）
4. 点击"创建新对话"提交

### 查询现有对话

1. 进入实验页面
2. 在"查询现有对话 ID"输入框中输入对话 ID
3. 点击"查询"按钮

### 发送消息

1. 创建或查询对话后
2. 在底部"发送消息"文本框中输入内容
3. 点击"发送"按钮

### 查看对话历史

- 创建或查询对话后，页面中间部分会显示对话历史
- 用户消息和 AI 消息有不同的样式区分
- 支持复制消息内容
